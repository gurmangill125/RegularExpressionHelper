Regular Expression Helper

  • \d: Matches any digit (0-9)
  • \D: Matches any non-digit character
  • \w: Matches any word character (a-z, A-Z, 0-9, _)
  • \W: Matches any non-word character
  • \s: Matches any whitespace character (space, tab, newline)
  • \S: Matches any non-whitespace character
  • [abc]: Matches any character a, b, or c
  • [^abc]: Matches any character except a, b, or c
  • (abc): Grouping: Matches the sequence "abc"
  • a|b: Alternation: Matches either a or b
  • a*: Zero or more occurrences of a
  • a+: One or more occurrences of a
  • a?: Zero or one occurrence of a
  • a{3}: Exactly 3 occurrences of a
  • a{2,4}: Between 2 and 4 occurrences of a
  • ^abc: Anchors to the start of a line
  • abc$: Anchors to the end of a line
  • \bword\b: Word boundary: Matches the whole word "word"
  • \Bword\B: Non-word boundary: Matches "word" within a word
